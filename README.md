# Miscelánea de Métodos Numéricos II


## Contenido del Proyecto
- `datos.csv`: Matriz de datos equidistantes inicial.
- `resultados_totales.csv`: DataFrame final exportado con Pandas.
- `analisis_datos.py`: Algoritmos de análisis usando librerías de datos.
- `metodos_poo.py`: Implementación bajo el paradigma de Programación Orientada a Objetos.

## Instrucciones de ejecución
Para correr los análisis, ejecuta en tu terminal:
python analisis_datos.py
python metodos_poo.py
